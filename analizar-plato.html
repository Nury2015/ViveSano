Para activar el análisis con IA necesitas desplegar la Firebase Function:
1. Instala Firebase CLI: npm install -g firebase-tools
2. Login: firebase login
3. Guarda la API key: firebase functions:secrets:set ANTHROPIC_API_KEY
4. Despliega: firebase deploy --only functions
